Part 1 - HTML

In this part of the workshop, you will learn what HTML is and how HTML is structured.

1.1 - HTML Notes

  • All webpages are written in HyperText Markup Language (HTML)
    • Either dynamically produced or statically written ahead of time
  • HTML consists of content enclosed in nested, predefined tags
    • Tags define the structure and role of content
    • Ex: <p>Hi there</p> tells the browser that “Hi there” is a paragraph
    • No one is expected to know every tag
      • Learn a handful of the most common tags, lookup others as you go
      • www.w3schools.com/tags
